## 📌 이 프로그램은 뭔가요?

**베이커리란 AI 매니저**는 빵집 사장님을 위한 **AI 비서 웹사이트**예요.

사장님이 오늘 있었던 일을 **한 줄만** 적으면,  
AI가 자동으로 **3개의 SNS 채널**(스레드, 인스타그램, 당근마켓)에 올릴 글을  
**동시에 뚝딱** 만들어줘요! 🎉

게다가 **구글에서 지금 사람들이 많이 검색하는 키워드(트렌드)**를  
자동으로 찾아서 글에 살짝 넣어줘요. 특히 제주도나 디저트/빵과 관련된 트렌드를 우선적으로 골라줍니다!

## 🖥️ 시작하는 방법 (처음 한 번만 하면 돼요!)

> **💡 중요:** 맥북(Mac) 기준 설명과 윈도우(Windows) 기준 설명이 조금 달라요! 본인의 컴퓨터에 알맞은 명령어를 사용해 주세요.

### 1단계: API 키 설정하기

AI가 글을 쓰려면 구글의 허가증(=API 키)이 필요해요.

1. [Google AI Studio](https://aistudio.google.com/) 에 접속해요
2. 로그인한 뒤 **"Get API Key"** 버튼을 눌러서 키를 복사해요
3. `app/backend/.env` 파일을 열어서 아래처럼 수정해요:

```
GOOGLE_API_KEY=여기에_복사한_키를_붙여넣기
```

### 2단계: 프로그램 실행하기

터미널(검은 화면)을 **2개** 열어야 해요. 맥은 `터미널` 앱, 윈도우는 `명령 프롬프트(cmd)`나 `PowerShell`을 여세요.

**터미널 1번** (AI 두뇌를 켜는 것):
*먼저 `베이커리란/app/backend` 폴더로 이동한 뒤 실행해요.*

**🍏 맥(Mac) 사용자:**
```bash
cd app/backend
source venv/bin/activate
uvicorn main:app --reload --port 8000
```

**🪟 윈도우(Windows) 사용자:**
```cmd
cd app\backend
.\venv\Scripts\activate
uvicorn main:app --reload --port 8000
```
> *(주의: 윈도우에서는 `(venv)`라는 글자가 앞에 떠야 정상입니다!)*

**터미널 2번** (웹 화면을 켜는 것):
*새 터미널을 열고 `베이커리란/app/frontend` 폴더로 이동한 뒤 실행해요.*

```bash
cd app/frontend
npm run dev
```

### 3단계: 웹사이트 열기

브라우저(크롬, 사파리 등)를 열고 주소창에 입력:
```
http://localhost:5173
```

짜잔! 🥐 베이커리란 AI 매니저가 나타나요!

## 🔥 기능 1: SNS 글 동시 생성

이것이 **메인 기능**이에요!

### 사용 방법:
1. 상단 탭에서 **⚡ SNS 글 생성**을 클릭해요
2. 왼쪽 입력창에 오늘 있었던 일을 자유롭게 적어요
   - 예: `비가 부슬부슬 오는 아침, 소금빵이 영롱하게 나왔어!`
3. **"SNS 초안 동시 생성하기"** 버튼을 클릭!
4. 잠깐 기다리면 (뼈대 모양 로딩이 나와요!) 3개의 카드가 짠! 하고 나타나요:
   - 🧵 **스레드** 글 (짧고 감성적)
   - 📸 **인스타** 글 (해시태그 포함, 비주얼 중심)
   - 🥕 **당근마켓** 글 (동네 주민한테 말 거는 느낌)

### 글 복사하기:
각 카드 오른쪽 위에 **"복사"** 버튼이 있어요.  
누르면 글이 클립보드에 복사되고, **"복사됨!"** 이라고 바뀌어요.  
그대로 SNS 앱에 **붙여넣기(Ctrl+V)** 하면 끝! ✨

## 📅 기능 2: 주간 콘텐츠 캘린더

한 주치 SNS 계획을 한 번에 세울 수 있어요!

### 사용 방법:
1. 상단 탭에서 **📅 주간 캘린더**를 클릭해요
2. 이번 주 특이사항을 적어요
   - 예: `수요일 신메뉴 말차 소금빵 출시, 토요일 관광객 많을 예정`
3. **"주간 콘텐츠 캘린더 생성"** 버튼 클릭!
4. 몇 초 뒤 **월~일요일** 각 요일별로 3개 채널의 포스팅 아이디어가 카드로 나타나요

## 🌙 기능 3: 다크모드

밤에 눈이 편하도록 **어두운 화면**으로 바꿀 수 있어요!

- 오른쪽 위 **달 모양(🌙)** 버튼을 누르면 → 다크모드 ON
- 다시 **해 모양(☀️)** 버튼을 누르면 → 라이트모드 ON
- 브라우저를 껐다 켜도 설정이 유지돼요! (기억해놓거든요 😎)

## 📊 구글 트렌드는 어떻게 반영되나요?

글 생성 버튼을 누르면, AI가 자동으로 이런 일을 해줘요:

1. 구글에서 **지금 한국에서 뜨고있는 검색어 15개**를 가져와요
2. 그중에서 **제주도, 빵, 디저트, 카페** 등과 관련 있는 키워드를 **먼저 골라요**
3. 골라진 트렌드 키워드를 AI 글쓰기에 **슬쩍 반영**해요
4. 화면 상단에 **"오늘의 트렌드"** 태그로 어떤 키워드가 사용됐는지 보여줘요

> 💡 **팁**: 트렌드가 빵집과 안 어울리면 AI가 억지로 넣지 않아요! 자연스러울 때만 넣어요.

## ❓ 자주 묻는 질문 (FAQ)

### Q: 버튼을 눌러도 아무 일이 안 일어나요!
**A:** 터미널 1번(백엔드)이 제대로 실행 중인지 확인해주세요.  
`uvicorn main:app --reload --port 8000` 이 실행되고 있어야 해요.

### Q: "API Key missing" 에러가 떠요!
**A:** `app/backend/.env` 파일에 진짜 API 키를 넣었는지 확인해주세요.  
키를 넣은 후 터미널 1번을 **껐다가 다시 켜야** 적용돼요!

### Q: 글이 마음에 안 들어요. 다시 만들 수 있나요?
**A:** 네! 그냥 **버튼을 다시 누르면** AI가 새로운 글을 만들어줘요. 매번 다르게 나와요!

### Q: 인터넷이 없어도 되나요?
**A:** 아니요, AI(Gemini)와 구글 트렌드를 사용하기 때문에 **인터넷 연결이 필요해요**.

## 💻 윈도우 PC로 프로젝트 옮기기

만약 제주도 매장의 윈도우 랩탑으로 이 프로그램을 옮기고 싶다면, 다음 순서대로 진행하세요!

### 1단계: 준비물 설치 (새 윈도우 PC에)
1. **Python 설치**: [python.org](https://www.python.org/downloads/)에 접속해서 파이썬을 받아요. **설치할 때 반드시 `Add Python to PATH` 박스에 체크**해야 해요!
2. **Node.js 설치**: [nodejs.org](https://nodejs.org/ko/)에 접속해서 `LTS 버전`을 다운받고 설치해요. (그냥 다음, 다음 누르면 돼요)

### 2단계: 폴더 복사하기
1. 맥북에 있는 `베이커리란` 폴더를 USB 메모리나 구글 드라이브 등을 이용해 **윈도우 PC의 바탕화면**으로 그대로 복사해요.
   > ⚠️ 단, `app/backend/.env` 파일(API 키 들어있는 숨김파일)이 제대로 복사됐는지 확인하세요! 만약 안 보인다면 윈도우에서 똑같이 다시 만들어주면 됩니다.

### 3단계: 윈도우에서 처음 세팅하기 (한 번만)
윈도우에서는 라이브러리들을 윈도우용으로 다시 다운받아야 해요!

**가. 프론트엔드 세팅:**
1. 윈도우에서 `명령 프롬프트(cmd)`를 열어요.
2. `cd Desktop\베이커리란\app\frontend` 입력 후 엔터
3. `npm install` 입력 후 엔터 (기다리면 라이브러리가 설치돼요)

**나. 백엔드 세팅:**
1. 명령 프롬프트에서 `cd ..\backend` 입력해서 백엔드 폴더로 이동해요.
2. `python -m venv venv` 입력 (가상환경 만들기)
3. `.\venv\Scripts\activate` 입력 (가상환경 켜기)
4. `pip install -r requirements.txt` 입력 (AI 두뇌 지식 다운로드)

이제 세팅이 끝났어요! 앞의 **'2단계: 프로그램 실행하기 (윈도우 사용자)'** 방법대로 터미널 2개를 띄워서 실행하시면 빵집에서도 윈도우로 편하게 쓰실 수 있습니다! 🍞

## 🗂️ 파일 구조 (궁금한 사람만!)

```

├── backend/                ← AI 두뇌 (Python)

    ├── src/

    │   └── index.css        ← 예쁜 디자인 코드
    └── index.html           ← 웹페이지 뼈대
```
